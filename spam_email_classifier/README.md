# Spam Email Classifier

## Description
This project uses Machine Learning to classify SMS messages as Spam or Not Spam.

## Features
- Upload SMS dataset
- Train a Naive Bayes model
- Predict spam messages
- Display model accuracy

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
SMS Spam Collection Dataset

## Accuracy
98.39%
